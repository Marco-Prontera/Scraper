# Scraper

### Installing and use

run docker and build and run container:

```
 1) bash docker-console/build.sh
```

connect to container shell

```
 2) bash docker-console/symfony.sh
```

composer install

```
 3) composer install
```

run command

```
 4) bin/console ads-scraper
 example:
```
